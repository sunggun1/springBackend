# springBackend
스프링, 리액트, aws를 사용한 메모 프로젝트 입니다.<br>
backend : spring datajpa, jpa, lombok, spring security, jwt token<br>
frontend : react <br>
server : ec2, rds<br>
sql : mysql, h2
<hr>

서버 주소들<br>
aws backend elastic ip : http://13.124.30.40:8080<br>
aws frontend elastic ip : http://3.39.145.245:3000<br>
